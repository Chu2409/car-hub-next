## Tecnologías

- TypeScript
- React
- Next.js

## Requisitos

- Node.js ^20.13.1
- Yarn

## Instalación

1. Instalar las dependencias de la aplicación

```bash
$ yarn install
```

2. Iniciar la aplicación:

```bash
# development watch mode
$ yarn dev

# production mode
$ yarn build
$ yarn start
```
